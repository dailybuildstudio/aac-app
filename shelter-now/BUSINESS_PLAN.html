01 · The one-pager

In three sentences

Problem

Florida residents in a housing crisis or facing evacuation can't find shelters fast — FEMA's app is rated 1.3 stars, no Spanish-first option exists, and 44% of Katrina evacuation refusals cited pets.

Solution

Find Shelter Now is a free, anonymous, offline-capable PWA showing real-time hurricane shelters and a year-round homeless directory. Eiren is the operator dashboard that keeps the data accurate in 10-second updates.

Wedge

Spanish-first + hurricane-tuned + offline-first + Florida-specific + zero-PII. No competitor hits all five.

05 · The disaster wedge

Designed for the moment the network is gone.

Cell networks routinely saturate before landfall and stay down for hours-to-days after. This is where Find Shelter Now beats every competitor.

📡
Pre-storm priming
"Save shelters for offline" CTA caches full county directory + map tiles before signal drops.
📞
tel: links work offline
Phone numbers function with no data — only signal. Every shelter card is callable.
💾
IndexedDB + Service Worker
Last-known FL shelter list cached on device. Opens with no signal. Refreshes when back online.
🔋
Battery-saver mode
Dim theme, no animations, lightweight rendering — preserves dwindling battery during outages.
🔄
Background sync (Eiren)
Operator updates queued offline, push when signal returns. Shelter staff with bad WiFi still publish.
🛟
Degraded fallback
If FL state portal is down, public site auto-falls back to Eiren operator data. Defense in depth.

Sell unified system to county and state Emergency Management agencies as their official shelter platform.

09 · Risks + mitigations

What could break this.

Wrong shelter data costs trust
Disclaimer + 211 fallback on every result. "Last updated" timestamp visible. Conservative status interpretation.
FL portal HTML changes break the scraper
Worker has try/catch + "data temporarily unavailable, call 211" fallback. Daily health-check ping. Eiren operator data as further fallback (v1.5+).
Spanish translation tone is off
One bilingual reviewer pass before launch. Hispanic Access Foundation partnership is the cleanest source.
Crowded space (Shelter App, ShelterBridge, FindHelp)
Win on the wedge: Spanish-first + hurricane-tuned + offline-first + Florida depth. Don't try to be national or generic.
Eiren onboarding gets gamed (fake facility)
Multi-step verification: facility ID → org domain email → callback to listed phone → manual approval for first 100. Auto-verify against HUD CoC + FL DEM registries thereafter. Address-poisoning blocked: address field is read-only via self-service.
Liability for publishing shelter info
Aggregating PUBLIC data with attribution; disclaimer + "verify via 211" is the standard of care. ToS disclaiming warranty before launch.
